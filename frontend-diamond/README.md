# FastRevEd Kids - Plateforme Éducative

## Structure du Projet

- **backend/** - API et serveur
- **frontend/** - Interface 9-12 ans  
- **frontend-diamond/** - Interface 6-8 ans
- **docs/** - Documentation
- **scripts/** - Scripts de test

## Démarrage Rapide

### Backend
```bash
cd backend
npm install
npm start
```

### Frontend 9-12 ans
```bash
cd frontend
npm install
npm start  # PORT 3000
```

### Frontend Diamond 6-8 ans
```bash
cd frontend-diamond
npm install
npm start  # PORT 3001
```

## Ports
- Backend: 5000
- Frontend (9-12 ans): 3000
- Frontend Diamond (6-8 ans): 3001
